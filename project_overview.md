# Project Overview

## Project Title
Simple Python Calculator

## Objective
The objective of this project is to build a basic arithmetic calculator using Python and to learn essential development tools such as Visual Studio Code, Git, and GitHub. The project aims to provide practical experience in coding, documentation, and version control workflows.

## Description
This is a menu-driven Python calculator that performs:

- Addition  
- Subtraction  
- Multiplication  
- Division (with divide-by-zero protection)

The user selects an operation from the menu, enters two numbers, and the program displays the result.  
The calculator continues to run until the user chooses to exit.

## Purpose of the Project
- Strengthen programming fundamentals  
- Understand development workflow  
- Learn Git & GitHub  
- Practice documentation
